# Tong Dev — Portfolio

React + Vite + Tailwind + React Router

## Quick Start
```bash
npm install
npm run dev
```

## Personalize
1. **Profile photo** → put `profile.jpg` in `public/`, then set `PROFILE_IMAGE = '/profile.jpg'` in `Home.jsx` and `About.jsx`
2. **GitHub links** → search `YOUR_GITHUB_USERNAME` and replace everywhere
3. **Your name** → search `Tong` and replace with your real name
4. **Email** → set `YOUR_EMAIL` in `Contact.jsx`
5. **Projects** → edit the `projects` array in `Home.jsx`
6. **Bio / timeline** → edit `About.jsx`

## Deploy to Vercel
Push to GitHub → [vercel.com](https://vercel.com) → Import → Deploy ✅
(vercel.json handles React Router redirects)
